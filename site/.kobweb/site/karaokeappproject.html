Karaoke App

Côté client:

  • Le produit: Application android native
  • Outils utilisés: Le langage Kotlin et le framework Jetpack Compose
  • Disponibilité: Sur le play Store, uniquement dans la zone géographique Guadeloupe
  • Partage du code: Code source disponible sur github

Côté serveur:

  • Le produit: Serveur dédié à l'administrateur déployé sur le web.
  • Outils utilisés: Le langage Kotlin et le framework Ktor.
  • Disponibilité: via un url avec des identifiants.
  • Partage du code: Code source disponible sur github.
